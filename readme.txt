Payment Fees for WooCommerce allows you to add custom fees for payment methods to your Products

== Description ==
Payment Fees for WooCommerce allows you to add custom fees for payment methods to your Products.
This has the huge advantage for you, that you no longer need to include the fees of your pamyent providers into your products.


== Installation ==
1. Extract payment-fees-for-woocommerce.zip
2. Upload the plugin folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
